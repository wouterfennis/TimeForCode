# Glossary

Status: Current

This glossary covers general project and domain terms used in the TimeForCode documentation. For architecture-specific terms, see [Arc42 Section 12 — Glossary](../architecture/arc42/12-glossary.md).

---

| Term | Definition |
| --- | --- |
| Campaign | An admin-curated group of projects with a shared theme (e.g. security, accessibility, documentation). Used to help donors discover relevant projects. |
| Commitment | The total number of developer hours pledged in a single donation. |
| Completion Rate | The ratio of hours logged to hours committed across all donations for an organisation or project. A measure of follow-through. |
| Donor | A company (DonorOrganization) or individual (Contributor) that pledges developer hours to a project. |
| Feature Request | A type of open task a project maintainer may define to attract specific contributions. |
| Hours Available | The pool of developer hours an organisation or individual has committed to donate in a given period. |
| Hours Spent | The cumulative hours actually logged across all completed donation transactions. |
| Impact Report | A summary document showing the total hours pledged, logged, and projects supported by a donor or received by a project. |
| Matching | The process of connecting a donor's available hours and skills to a project's stated needs. Can be manual (donor browses and chooses) or assisted (platform suggests). |
| Open Source Project | A software project with a publicly accessible source code repository (in the current platform, hosted on GitHub) that is listed on TimeForCode to attract contributions. |
| Pledge | The act of committing hours to a project. Creates a Donation record in the platform. |
| Platform Administrator | A trusted operator of the TimeForCode platform. Responsible for approving projects, moderating content, and managing user issues. |
| Project Listing | The public page on the TimeForCode website showing a project's description, needed skills, current donations, and progress. |
| Recognition | A visible acknowledgement of a donor's contributions. Currently planned as badges and tiers. |
| Spotlight | A promoted position on the website homepage or in a campaign, highlighting a specific project. |
| Tech Stack | The set of programming languages, frameworks, and infrastructure used to build a project. Used for matching donors to projects. |
| Time Donation | Informal synonym for an hour-based donation pledge. |
| Transaction | See DonationTransaction. |
